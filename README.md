# 🎭 Stage Schedule Manager

A web application to manage **event schedules, stage allocations, performers, and crew tasks** for live events.

This system helps event organizers avoid **schedule conflicts, delays, and miscommunication** during performances.

---

## 🚀 Tech Stack

Frontend:
- React
- Bootstrap
- Axios

Backend:
- Node.js
- Express.js

Database:
- MongoDB

Authentication:
- JWT
- bcrypt

---

## 👥 User Roles

1. Admin
2. Stage Manager
3. Performer
4. Crew Member

Each role has different permissions within the system.

---

## ✨ Features

- User Authentication (Login / Register)
- Event Management
- Stage Creation
- Performer Scheduling
- Crew Task Management
- Conflict Detection
- Dashboard Overview
